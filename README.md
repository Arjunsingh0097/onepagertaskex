# One Page Taskex Website

A modern, responsive one-page website for Taskex with a black theme.

## Features

- ✅ Fully responsive design
- ✅ Dark theme with teal accents
- ✅ Interactive world map visualization
- ✅ Smooth scrolling animations
- ✅ Mobile-friendly navigation
- ✅ Optimized performance

## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/Arjunsingh0097/onepagertaskex.git
cd onepagertaskex
```

2. Open `index.html` in your browser, or use a local server:

```bash
# Using Python
python3 -m http.server 8000

# Using Node.js (after npm install)
npm start
```

3. Open your browser and navigate to `http://localhost:8000` (or `http://localhost:3000` if using npm start)

## Deployment

### GitHub Pages

1. Push your code to GitHub
2. Go to repository Settings > Pages
3. Select source branch (main) and folder (root)
4. Your site will be available at `https://Arjunsingh0097.github.io/onepagertaskex/`

### Other Hosting Options

- **Netlify**: Drag and drop the folder or connect your GitHub repo
- **Vercel**: Connect your GitHub repository
- **AWS S3**: Upload files to an S3 bucket with static website hosting
- **Any web server**: Upload all files to your web server's root directory
